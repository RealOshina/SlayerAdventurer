# God Mod
This mod is from SlayerAdventurer. With this mod, you can use god mod which mean you can't die and spawn any item you want.

Made by [RealOshina](https://github.com/RealOshina)
## Installation
Put it in the `mods` folder and you ready to go!

## Commands
To enable god mod in game, use this command `/mods god --enable` or if it said something like:
> [ERROR] The game cannot find the 'god' mod. Please try check the mod folder again.

or 
> [ERROR] There is something wrong with mod.

use this commad `/mod god --find` and identify that there is god mod folder and only one of the folder it can find. It there is none, please reinstall the file again. Or if there is two god mod, use `/mod god --delete 1`.

Then try `/mod god --enable` again, if it still fail. You can try install the mod and the game again.

The command lists:
| Commands | What it does |
|----------| -------------|
|/mod god set god-mode| Set you to god mod which mean nothing can hurt you |
|/mod god set normal-mode| Set you to normal mod.|
|/mod god spawn *\<item_id>*| Give you an item based on item id. You can see it at SlayerAdventurer in official SlayerAdventurer github page. [Official SlayerAdventurer Github](https://github.com/RealOshina/SlayerAdventurer)|
|/mod god --test | Check if mod work correctly|
